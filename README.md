GTG Perfumes

🚀 Live Demo: [https://gtg-perfumes-frontend-rho.vercel.app/](https://gtg-perfumes-frontend-rho.vercel.app/)

A pixel-perfect, responsive frontend implementation for an e-commerce perfume website. This project was developed by converting a professional Figma design into a fully functional website using vanilla HTML, CSS, and JavaScript, without relying on external frontend frameworks.

🌟 Features

* **Responsive Design**: Fluid layout that adapts seamlessly to laptop, tablet, and mobile viewing experiences.
* **Dynamic Product Gallery**: Interactive gallery interface for browsing perfume products.
* **Interactive Subscriptions**: Expandable subscription sections with conditional logic for the "Add to Cart" flow.
* **Scroll-Triggered Animations**: Engaging count-up animations for the statistics section that trigger as the user scrolls into view.
* **Mobile Navigation**: A clean and responsive hamburger menu tailored for smaller viewports.
* **Comparison Table**: A detailed and accurately styled "Why GTG is the #1 Choice" responsive comparison table.
* **Zero Dependencies**: Built entirely with plain HTML, CSS, and JavaScript for optimal performance and deeply customized styling.

🛠️ Technologies Used

* **HTML5**: Semantic markup for clean document structure and accessibility.
* **CSS3**: Custom styling, modern Flexbox layouts, and CSS variables for a maintainable design system. Complete mobile responsiveness achieved via Media Queries without hardcoded pixel widths.
* **JavaScript (Vanilla/ES6+)**: DOM manipulation, event handling, cart conditional logic, and scroll animations.

📁 Project Structure

🚀 Getting Started

To view the project locally, no complex build tools or development servers are required.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/VIPUL162005/GTG-Perfumes-Frontend.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd GTG-Perfumes-Frontend
   ```
3. **Run the application**:
   Simply open the `index.html` file in your preferred web browser. Alternatively, use an extension like **Live Server** in VS Code for an optimal development experience with hot reloading.

🎨 Design Philosophy

This project represents a high-fidelity implementation of a UI/UX Figma design. Special attention was paid to:
* Pixel-perfect accuracy regarding dimensions and margins.
* Maintaining a clean, comment-free codebase for identical production deployment.
* Ensuring the layout remains robust and visually appealing down to mobile screen sizes without horizontal overflow.
